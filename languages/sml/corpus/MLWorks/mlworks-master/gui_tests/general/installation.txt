Request 30017 (Unix).

Check the names of the executable files in the bin directory.  They should
include the OS and architecture of the platform (and no other platforms!).

Check that the directory lib/X11/nls exists in the distribution, with
contents that match an X11R5 machine.


Request 20030 (Windows).

Check that the installation adds icons for the User Guide and the
Reference Manual.
